The general titlebar options affect the titlebar both in active and inactive state.

Enable titlebar enhancements enables/disables the enhanced titlebar customization features.
When this option is unchecked the Active and Inactive titlebar pages will not be accessible.

When Override presentation parameters is unchecked, titlebars which do not use default colors for the active or inactive state will ignore the Styler titlebar background settings.

The Font control shows the name of the current titlebar font. The ellipsis button display the default font dialog allowing to select a different font.

Via the Text alignment drop down list is possible to center or left align the titlebar text.
