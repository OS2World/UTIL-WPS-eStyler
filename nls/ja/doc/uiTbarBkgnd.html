The controls displayed in this group of settings vary according to what is currently selected in the Type of background drop down list.

Solid color background

When Solid color is selected there is only one sub-option: the Background color button which displays a standard color wheel control to let the user choose the titlebar background color.

Gradient background

More options are available when the selected background type is Gradient.

The Gradient direction drop down list selects the kind of gradient: Horizontal, Vertical, Horizontal - double, Vertical - double.
The preview window makes the distinction among these different styles clearer than any possible descriptions as it is dynamically updated according to the current selection.

The text of the two color buttons below the Gradient direction drop down list varies according to the current gradient style and should be self-explicative. In case of doubt the user can just experiment and see the result in the preview window.

The Swap colors button swaps the two gradient colors.

Bitmap background

Bitmap is the last available background type.

The Background image drop down lists the images available in the Styler bitmaps folder.
The preview window is dynamically updated as a new bitmap file is selected.

The Add a new image button allows to import a new bitmap file into the Styler bitmaps folder via the standard file dialog.
Valid bitmap files are OS/2 bitmaps of any format and color depth provided that their size is less than 64 KB.
If the bitmap file is an array of bitmaps only the first bitmap is taken into account.

The Scale the image checkbox toggles the image rendering mode between tiling and scaling.
