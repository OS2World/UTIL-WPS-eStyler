eStyler for eComStation improves both the aesthetics and the behavior of some OS/2 Presentation Manager controls and of the eComStation system in general.

The Shutdown settings notebook controls the enhancements of the system shutdown.
The new confirmation dialog allows to perform various kind of power management or system shutdown, associating with the event the execution of a list of programs, the automatic termination of those programs not proprely responding to the system shutdown request, and allowing to reboot the system optionally choosing a different boot volume.

Enable extended shutdown activates the enhanced shutdown features. When this option is checked all the depending dialog controls and notebook tabs get enabled.

The Include the following shutdown options group provides a list of actions optionally displayed in the shutdown dialog.

When Animate the shutdown dialog is checked an animation, gradually darkening the screen, introduces the shutdown confirmation dialog.
The Animation steps spin button controls the animation speed: the more the steps the longer the time taken by the animation.
