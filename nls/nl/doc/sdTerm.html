On system shutdown some programs do not behave correctly and just refuse to quit so, to let the system being correctly closed, it is necessary to use an appropriate utility to close them (in such case the utility should be entered in the Shutdown programs list) or to recur to the Termination list feature to terminate them via the DosKillProcess() API.

The Terminate the following programs on shutdown checkbox enables/disables the termination of the programs listed below.

The list of programs to be terminated, by default contains:

  • CHAT.EXE
  • CLKBASIC.EXE
  • CAD_POP.EXE
New program names can be added by entering them in entry field below the list and clicking on the Add button.

Remove deletes the selected program from the list.

When Edit is pressed, the selected program name is entered in the entry field to let the user modify it. Once the program name has been modified the Add button must be pressed again to replace the name of the selected program with the new entry field content.

Find opens the standard file dialog allowing the user to find and select an executable name.

Apply saves the termination list to the eStyler.ini file, while Undo restores the list to the state it was in when the settings notebook was opened.
