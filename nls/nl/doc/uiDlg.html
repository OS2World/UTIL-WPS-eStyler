Usually, when no other kind of font has been explicitly specified by the programmer, all dialog windows use the ugly System Proportional font.

eStyler allows to override this system behaviour forcing all dialog windows to use the user's favourite font.
When the Set a default font for the dialog windows checkbox is checked all dialog windows will use the font selected via the ellipsis button below.
This will work in all dialog windows which use the system default font. Override Presentation Parameters forces all dialog windows to use the selected font.

Note:
The dialog controls inherit the font attribute (presentation parameter) from the dialog unless they have been explicitly set to use a different font. In this latter case, the eStyler font feature cannot work.
