The border settings are quite simple. The Show the titlebar border checkbox selects if the titlebar has to be surrounded by a border.

Two different colors are used for the titlebar border in order to achieve a 3D visual effect. The Left/top color and the Right/bottom color buttons allow to choose what color to use for that purpose. While the Swap colors button changes the effect from depressed to raised according to the currently selected colors.
Of course is always possible to avoid the 3D effect by setting the same color in both buttons.

The preview window, as usual, is dynamically updated according to the selected options.
