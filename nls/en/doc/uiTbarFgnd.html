The Text shadow checkbox provides the text with a sort of shadow of an user selectable color to get a more outstanding titlebar text.

The Text color and the Text shadow color buttons allow the user to fully customize the titlebar text.
The Swap colors button swaps the two colors allowing to change with one click the look of the text from depressed to raised.
