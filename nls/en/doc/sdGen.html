Styler improves both the aesthetics and the behavior of some OS/2 Presentation Manager controls and of the system in general.

The Shutdown settings notebook controls the enhancements of the system shutdown.

The enhanced confirmation dialog allows you to perform various kinds of power management or system shutdown. You can define a list of programs to be executed on shutdown, maintain a list of processes to be terminated automatically (for certain programs which do not respond properly to the system shutdown request), and optionally enable rebooting the system to a specific boot volume.

Enable extended shutdown activates the enhanced shutdown features. When this option is checked all the depending dialog controls and notebook tabs get enabled.

The Include the following shutdown options group provides a list of actions optionally displayed in the shutdown dialog:

  • When Animate the shutdown dialog is checked, an animation will introduce the shutdown confirmation dialog by gradually darkening the screen.
  • The Animation steps spin button controls the animation speed: the more steps, the longer the time taken by the animation.
  • When Capture mouse pointer is checked, the mouse will be prevented from moving outside the shutdown options dialog. Enabling this option has been known to cause problems with some remote desktop control utilities.
