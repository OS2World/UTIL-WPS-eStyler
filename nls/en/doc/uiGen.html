Styler improves both the aesthetics and the behavior of some OS/2 Presentation Manager controls and of the system in general.

The User interface object controls the look of the title bars and push buttons and the font used by the dialog windows.

If one still prefers the old warp 4 style, it is possible to just uncheck the Enable user interface enhancements checkbox or individually disable the titlebar, button or dialog enhancements.

Besides that, the advanced look provided by eStyler can be selectively disabled for a given set of applications.
Use the entry field above the Add, Remove and Find buttons to enter the name of an executable file and the Add button to add the new entry to the list.
The Remove button removes the selected entries from the list.
The Find button displays the standard file dialog and allows to directly add the selected file name to the list.

Once the list has been modified, the Apply button gets enabled allowing to save the new entries and make eStyler aware of the changements.

Note:
Enabling/disabling the user interface enhancements is performed as soon as the state of the checkbox is changed: there is no need to use the Apply button for that purpose.

The common buttons documentation provides some further informations about the Apply button and the other buttons displayed in the common buttons area of the settings notebook.
