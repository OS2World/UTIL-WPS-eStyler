Styler improves both the aesthetics and the behavior of some OS/2 Presentation Manager controls and of the system in general.

This Windows keys extends the keyboard functionality allowing the user to assign some useful functions to the keyboard Windows keys. The assigned functions are performed as soon as the key is released.

Do not perform any action
This just disables the key.
Hide/show all windows
This hides/unhides all windows visible in the current desktop (just in case you are running a virtual desktop utility), allowing you to access the desktop icons.
Show window list
This function displays the window list at the current mouse position.
Show context menu
If the focus window has any context menu, it will pop up as the selected key is released.
Lock up the desktop
Locks the desktop like the corresponding item of the desktop popup menu.
Open the Drives folder
This opens the Drives folder. If the folder is already opened, then it gets the focus and is brought to the top.
Open the System Setup folder
This opens the System Setup folder. If the folder is already opened, then it gets the focus and is brought to the top.
Open an OS/2 command window
This opens an OS/2 command window.
Show WarpCenter desktop menu
This opens the WarpCenter menu which displays the current desktop content. This and the following features will work only if the WarpCenter is currently running.
Show WarpCenter window list
The WarpCenter window list is displayed on the screen.
Show WarpCenter process list
If the statement
SET KILLFEATUREENABLED=ON
is present in your CONFIG.SYS file, then this list will enable you to kill processes.
Show WarpCenter tray list
This function displays the WarpCenter trays menu.
Switch to the next WarpCenter tray
The next WarpCenter tray is brought to the top as soon as the selected key is released.
Show WarpCenter information menu
This functions shows the content of the Assistance center  menu on the left side of the WarpCenter clock.
Show the XButton menu (top XCenter)
The X button menu is displayed if XCenter is installed at the top of the screen and contains the button.
Show the XButton menu (bottom XCenter)
The X button menu is displayed if XCenter is installed at the bottom of the screen and contains the button.
Show the XCenter tray list
Display the menu of the tray widget of XCenter.
Switch to the next XCenter tray
The next XCenter tray is brought to the top as soon as the selected key is released.
Show XCenter tray 1-10
Display the nth XCenter tray assuming that trays are counted from the top of the tray widget menu.
Move the XCenter behind/on-top-of the other windows
Cyclically moves the XCenter(s) behind or on top of the other windows
The Undo button restores the current settings to the state they were in when the settings notebook was opened.

The Default buttons sets all the settings to their default values.
