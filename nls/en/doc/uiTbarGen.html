The general titlebar options affect the titlebar both in active and inactive state.

Enable titlebar enhancements enables/disables the enhanced titlebar customization features.
When this option is unchecked the Active and Inactive titlebar pages will not be accessible.

When Override presentation parameters is unchecked, titlebars which do not use default colors for the active or inactive state will ignore the Styler titlebar background settings.

Normally, titlebars will use the system-wide font defined by the current desktop scheme. This can be configured using the OS/2 Scheme Palette. If no system-wide titlebar font is defined, you can set a titlebar font using Styler. In this case, the Font control will appear on this page, showing the selected font; the ... button next to it will bring up the font selection dialog.

It is possible to align the titlebar text to the left or right of the titlebar using the Text alignment drop down list
