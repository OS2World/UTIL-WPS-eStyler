Various buttons can be displayed in the common buttons area of the settings notebook.
The visibility and enabled state of the buttons is automatically updated according to the purpose of the top page and the current state of the various settings it contains.

The Apply button, when visibled and enabled, generally refers to all the settings contained in the current top page.
The online help provides some note to warn the user of the exceptions to this rule (e.g. the Do not apply enhancements to: group of controls in the cover page of the User interface object).

The Undo button is enabled only when the state of the options of the current page differs from the previous state, i.e. when the notebook was opened.

The Default button is enabled only when any value of the current settings differs from the default values.

The Preview button shows or hides the preview window in order to let the user tune the current settings to his/her likes before applying them.
The Preview window is used only in the User Interface settings notebook.
