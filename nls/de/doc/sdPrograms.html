Der erweiterte Systemabschluß ermöglicht die Ausführung einer Liste von Anwendungen bevor das System beendet wird.

Wenn Führen Sie die folgenden Programme beim Systemstart aus aktiviert ist, werden die unten aufgeführten Anwendungen nacheinander ausgeführt. Erst wenn die Ausführung des letzten Programms endet, werden die zuvor geöffneten Fenster geschlossen und der Systemabschluß durchgeführt.
Wenn die Programmliste leer ist, wird die Option schlicht ignoriert.

Hinweis:
Der Bestätigungsdialog für den Systemabschluß erlaubt es, diese Funktion über ein Kontrollkästchen Anwendungen beim Systemabschluß ausführen zeitweise zu deaktivieren oder zu aktivieren.

Die Programmnamen werden über das Eingabefeld Anwendung eingegeben.
Befindet sich ein Programm im PATH (d.h. in einem Verzeichnis, das in der in der Datei CONFIG.SYS gesetzten Umgebungsvariablen PATH aufgelistet ist), wird nur der Programmname benötigt, ansonsten ist die Angabe des vollständigen Programmpfades erforderlich.

Wenn keine Erweiterung angegeben wird, wird die Erweiterung .EXE automatisch an den Programmnamen angehängt.
Wenn es sich bei dem Programm um ein REXX-Skript oder eine Stapel-CMD-Datei handelt, wird der Name automatisch angepaßt in:
CMD.EXE /C Dateiname.cmd Optionen

Wenn die REXX- oder Batch-Datei andere Programme aufruft, muß sie dies synchron tun, d.h. nicht durch einen START- oder DETACH-Befehl, da sonst nicht erkannt werden kann, wann jedes Programm beendet wird, und einige Dateien möglicherweise nicht richtig gespeichert werden.

Optionale Programmparameter können über das Feld Parameter eingegeben werden. Dieses Feld muß leer gelassen werden, wenn das Programm keine Parameter benötigt.

Die Sichtbarkeit des Programms (normal, maximiert oder verdeckt) kann über die Dropdown-Box Fenster eingestellt werden.

Die beiden Pfeilschaltflächen ordnen den Inhalt der Programmliste an und verschieben das ausgewählte Element nach oben oder unten.
Sie werden bei Bedarf automatisch aktiviert/deaktiviert.

Mit der Schaltfläche Hinzufügen wird der Inhalt des aktuellen Eingabefeldes an das Ende der Liste angefügt.

Entfernen löscht das ausgewählte Element aus der Liste.

Bearbeiten füllt die Eingabefelder mit den Daten des ausgewählten Elements, damit Sie die Änderung vornehmen können.
Sobald die Daten geändert worden sind, wird die Schaltfläche Hinzufügen benutzt, um das gewählte Element durch die neuen Daten zu ersetzen.

Suchen öffnet den Standard-Dateidialog, damit der Benutzer einen ausführbaren Dateinamen wählen kann.

Anwenden sichert die Listendaten in der Datei STYLER.INI, während Rückgängig den Listeninhalt in den Zustand zurücksetzt, in dem er sich im Einstellungsnotizbuch befand.
