Hinweis:
Dieser Dialog ist nur für die Fehlersuche vorgesehen.

Durch den erweiterten Systemabschluß werden, gegenüber dem regulären, einige zusätzliche Funktionen verfügbar.
Ein ordnungsgemäßer Systemabschluß kann manchmal von den Verzögerungen abhängen, die für die Ausführung der Verbesserungen festgelegt wurden.
Falls das System die Abschaltung nicht erfolgreich abschließen kann, mag es notwendig sein, einige Verzögerungswerte zu ändern.
In den meisten Fällen wird ein fehlgeschlagener Systemabschluß ohnehin durch ein Programm verursacht, das nicht korrekt auf selbigen reagiert. Das hat zur Folge daß der gesamte Prozeß auf diesen einen wartet, und das System somit scheinbar hängt. In einem solchen Fall müssen Sie den Namen des verursachenden Programms ermitteln, und ihn in die Liste zum automatischen Schließen eintragen.
Als Faustregel gilt: Wenn das System beim Systemabschluß hängt, selbst wenn die Abschaltverbesserungen deaktiviert wurden, dann liegt das Problem bei einem einzelnen Programm. Andernfalls liegt die Ursache in den Abschaltverbesserungen selbst.

Hinweis:
Alle Verzögerungen werden in Zehntelsekunden angegeben (z.B. ein Wert von 80 bedeutet 8 Sekunden).

Sperren und Ruhemodus
Verschiedene Hauptplatinen sind nicht vollständig kompatibel mit der fortschrittlichen OS/2-Stromverwaltung, so daß sich das System aufhängen kann, wenn ein Ruhe-Modus-Befehl ausgegeben wird, während die Maus bewegt wird. Die Sperr- und Ruhemodusverzögerung stellt sicher, daß das System vollständig angehalten ist, bevor es ausgesetzt wird.
Wenn Sie ein schnelleres Sperren und Aussetzen wünschen, versuchen Sie, eine geringere Verzögerung einzustellen, solange sich das System korrekt verhält.
Wenn Sie diese Funktion nicht verwenden, können Sie diese Einstellung ignorieren.

Ruhemodus
Wenn Ihr System hängt, nachdem Sie Ruhemodus gewählt haben, versuchen Sie, die Verzögerung auf 20-30 oder höher zu setzen.
Wenn Sie diese Funktion nicht verwenden, können Sie diese Einstellung ignorieren.

WarpCenter-Einstellungen sichern
Diese Verzögerung erlaubt es dem System, die WarpCenter-Einstellungen zu sichern. Wenn das System recht schnell ist, kann diese Verzögerung sicher auf 3-4 oder weniger gesenkt werden.
Wenn Sie das WarpCenter (SmartCenter) nicht verwenden, können Sie diese Einstellung ignorieren.

Fenster aufrollen
Diese Verzögerung ermöglicht das Aufrollen aller eingerollten Fenster beim Systemabschluß.
Ignorieren Sie diese Einstellung, wenn Sie das separate Styler/2-Produkt nicht installiert haben.
Wenn Sie die Styler/2-Rollfenster-Funktion nicht verwenden oder über ein schnelles System verfügen, können Sie diesen Wert sicher auf ein Minimum reduzieren.

Nachrichtenfenster beim Systemabschluß
Diese Verzögerung verhindert, daß die standardmäßigen Systemabschaltfelder angezeigt werden und ermöglicht die Ersetzung durch die verbesserten Dialoge.
Wenn sich das System beim Systemabschluß aufhängt, können Sie versuchen, diesen Wert durch Erhöhen oder Verringern zu ändern.

DosKillProcess
Diese Verzögerung steuert das Intervall, das die erweiterte Prozedur nach der Ausführung von DosKillProcess wartet, um alle Programme in der Abbruchliste zu stoppen, bevor sie WinShutdownSystem aufruft. Bei einigen Programmen kann es einige Augenblicke dauern, bis sie ordnungsgemäß beendet sind. Wenn dies bei Nicht-PM-Programmen der Fall ist, können diese den Prozeß des Systemabschluß stören. Möglicherweise müssen Sie diesen Wert erhöhen, wenn das selbiger fehlschlägt, weil ein oder mehrere Programme nicht rechtzeitig beendet werden.

WinShutdownSystem
Dieser Punkt verzögert die Zerstörung des FensterlisteFensters, wodurch die Ausführung des WinShutdownSystem API ausgesetzt wird.
Wenn das System beim Abschluß hängen bleibt, versuchen Sie, diesen Wert zu reduzieren.
Wenn das System seine Einstellungen oder den Desktop-Status beim Ausschalten oder Neustart nicht korrekt speichert, versuchen Sie, den Wert zu erhöhen.

DosShutdown
Diese Einstellung gilt nur, wenn Sie sich dafür entschieden haben, dem Bestätigungsdialog den regulären Systemabschluß (aufgeführt wie Abschalten) hinzuzufügen.
Dies ist die Zeit, die es zu warten gilt, bevor das Statusfenster angezeigt wird, daß darüber informiert wenn die Abschaltung abgeschlossen ist, und es sicher ist, das System abzuschalten.
Wenn Ihre Maschine schnell ist, können Sie diesen Wert auf das Minimum senken.

Abschalten
Diese Verzögerung stellt sicher, daß alle Systemeinstellungen und der Desktop-Status gespeichert werden, bevor das System abgeschalten wird.
Besitzer von schnellen Maschinen können diesen Wert senken.

Ok
Diese Schaltfläche speichert die aktuellen Einstellungen und schließt den Dialog.

Abbruch
Diese Schaltfläche schließt den Dialog, ohne die Einstellungen zu speichern.

Standard
Diese Schaltfläche setzt alle Verzögerungsoptionen auf die Standardwerte zurück.
