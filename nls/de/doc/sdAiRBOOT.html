AiRBOOT ist ein fortschrittlicher und funktionsreicher Bootmanager, der von Martin Kiewitz entwickelt wurde.

Damit die Neustart-Funktion mit AiRBOOT zusammenarbeitet, müssen die folgenden Anforderungen erfüllt sein:

  1. AiRBOOT Version 1.03 oder neuer muß installiert sein.
  2. SETABOOT.EXE muß als SETBOOT.EXE in ein Verzeichnis kopiert werden, das in der SET PATH-Direktive der Datei CONFIG.SYS vor dem Verzeichnis \OS2 aufgeführt ist, so daß es die Datei \OS2\SETBOOT.EXE umgeht.
    Alternativ kann \OS2\SETBOOT.EXE in SETBOOT.IBM umbenannt werden und SETABOOT.EXE kann als SETBOOT.EXE in das Verzeichnis \OS2 kopiert werden.
