Чекбокс Улучшение кнопок определяет, нужно ли применять в системе улучшения для кнопок. Когда флажок этого чекбокса установлен, доступны различные параметры для настройки внешнего вида всех кнопок.
  • Утолщённая рамка контролирует толщину границы кнопок.
  • Плоская рамка придаёт всем кнопкам уплощённый вид.
  • Переопределить цвета не по умолчанию заставляет все кнопки использовать цвет кнопки по умолчанию, игнорируя нестандартные цвета, установленные пользователем или программистами.
  • 3D-стиль кнопки по умолчанию изменяет акцент кнопок по умолчанию (то есть тех кнопок, которые активируются, как только пользователь нажимает клавишу Enter) со сплошной цветовой границы по умолчанию в 2 пикселя на двухцветную границу в 1 пиксель с 3D-эффектом нажатия.
  • 3D-стиль неактивной кнопки управляет внешним видом отключённых кнопок. Когда выбран этот параметр, текст отключённых кнопок окрашивается с трёхмерным эффектом, в противном случае отключённые кнопки просто окрашиваются серой текстурой.
  • Когда установлен флажок чекбокса Сплошной цвет фона фон кнопок окрашивается в сплошной цвет, а не как изображение.
    Когда флажок этого чекбокса снят, доступен другой набор элементов управления, позволяющий выбрать изображение для использования в качестве фона кнопки.
    В раскрывающемся списке Фоновое изображение перечислены изображения, доступные в данное время в папке растровых изображений Стайлера.
    Кнопка Добавить новое позволяет импортировать новый файл растрового изображения в папку растровых изображений Стайлера через стандартный файловый диалог.
    Допустимые файлы растровых изображений - это растровые 256-цветные изображения OS/2 любого формата меньше 64 КБ.
    Если файл растрового изображения представляет собой массив растровых изображений, учитывается только первое растровое изображение.
