Функция Программы выключения может использоваться для сохранения всех системных файлов. Для этого создайте текстовый файл со списком всех соответствующих системных файлов. Например (предположим, что D: Ваш загрузочный диск): 
d:\os2\*.ini
d:\desktop
d:\config.sys
d:\autoexec.bat
d:\os2\dll\*.cfg
После сохранения файла, например как FILES.LST, введите в поле Программа: 
CMD.EXE
и в поле Параметры: 
/C TYPE D:\FILES.LST | ZIP -rS9 D:\SYSTEM.SAV -@ 2>>D:\SYSSAV.LOG 1>>&2
Это создаст в корне диска D: ZIP-файл с именем SYSTEM.SAV, содержащий все файлы, указанные в D:\FILES.LST, а файл SYSSAV.LOG будет регистрировать весь вывод команды.
Теперь Вы можете создать простой файл CMD в корне загрузочного диска: 
@echo off
d:
cd\
unzip -o d:\system.sav
сохранив его, например, как RECSYS.CMD.
Если когда-либо случится так, что Вы не можете перезагрузить систему из-за повреждения системных файлов или WarpCenter потеряет свои данные конфигурации, Вам просто нужно загрузиться в командную строку и запустить RECSYS.CMD, чтобы восстановить предыдущую конфигурацию системы.

Примечание: Это не позволит сохранить конкретно последнее состояние системы, поскольку некоторые файлы (например, OS2.INI) не обновляются синхронно.
