Расширенное выключение позволяет выполнить список программ перед закрытием системы.

Если установлен флажок чекбокса Выполнить следующие программы при выключении перечисленные в списке программы выполняются одна за другой. Только по завершении выполнения последней программы ранее открытые окна закрываются, и система выключается.
Если список программ пустой, опция просто игнорируется.
Примечание:
Диалоговое окно подтверждения выключения позволяет временно отключить или подключить эту функцию с помощью чекбокса Выполнить программы выключения.

Имена программ вводятся через поле ввода Программа.
Если программа находится в PATH (т.е. находится в каталоге, указанном в переменной среды PATH, определённой в файле CONFIG.SYS), требуется только имя программы, в противном случае необходимо указать полный путь к программе.
Если расширение не указано, расширение .EXE автоматически добавляется к имени программы.
Если программа представляет собой сценарий REXX или пакетный файл CMD, её имя автоматически преобразуется в:
CMD.EXE /C filename.cmd параметры

Если REXX или пакетный файл вызывает другие программы, он должен будет сделать это синхронно, то есть не с помощью какой-либо команды START или DETACH, в противном случае невозможно будет определить, когда каждая программа завершается, и некоторые файлы могут быть сохранены некорректо.

Необязательные параметры программы можно ввести через поле ввода Параметры. Это поле необходимо оставить пустым, если программе не требуются параметры.

Видимость окна программы (Обычное, Мини или Макси) может быть установлена в раскрывающемся списке Окно.

Две кнопки со стрелками упорядочивают содержимое списка программ, перемещая выбранный элемент вверх или вниз. Они автоматически включаются/выключаются по мере необходимости.

Кнопка Добавить добавляет текущее содержимое полей ввода в конец списка.

Убрать удаляет выбранный элемент из списка.

При нажатии кнопки Редактировать данные для выбранного элемента вводятся в поле ввода, чтобы пользователь мог их изменить. После изменения данных необходимо снова нажать кнопку Добавить, чтобы заменить выбранные элементы новым содержимым поля ввода.

Найти открывает стандартный файловый диалог, позволяющий пользователю найти и выбрать имя исполняемого файла.

Применить сохраняет список с данными в файл STYLER.INI, а Откатить восстанавливает список до состояния, в котором он находился при открытии блокнота настроек.
