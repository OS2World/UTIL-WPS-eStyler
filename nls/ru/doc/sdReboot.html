С помощью функции Перезагрузки систему можно выключить, а затем автоматически перезагрузить с выбранного тома.

AiRBOOT поддерживается и IBM Boot Manager и Martin Kiewitz's.

Диалоговое окно страницы перезагрузки позволяет определить список томов для последующей перезагрузки, которые будут отображаться в диалоговом окне расширенного выключения.
Список томов можно заполнять вручную или автоматически.

Заполнить вручную

Поле ввода Имя тома должно быть заполнено тем значением, которое отображается в меню BootManager или AiRBOOT, а поле Описание тома позволяет ассоциировать с томом более содержательное (и длинное) описание.
В диалоговом окне подтверждения выключения в раскрывающемся списке отображаются только описания томов в том порядке, в котором они отображаются в списке Разрешить перезагрузку следующих томов.
Две маленькие кнопки со стрелками слева от списка позволяют изменять порядок элементов, перемещаясь вверх или вниз по выбранному элементу.

Кнопка Добавить становится доступной, после того, как оба поля ввода будут содержать текст. После нажатия кнопки содержимое полей добавляется в конец списка как:
описание тома (имя тома)
Кнопка Убрать удаляет элемент из списка.
Кнопка Редактировать позволяет изменить у выбранного элемента имя или описание тома. Если кликнуть мышью, то данные выбранного элемента помещаются в поля ввода, и кнопка Добавить становится активной, чтобы пользователь мог обновить выбранный элемент, после изменения содержимого полей ввода.

Заполнить автоматически

При нажатии кнопки Найти список загрузочных разделов считывается напрямую с диска, если установлен AiRBOOT, или через интерфейс LVM. Затем, если загрузочного тома ещё нет в списке, он добавляется в конец списка в следующем формате:
имя тома (имя тома)
то есть имя тома также используется в качестве описания тома, и пользователь должен будет использовать кнопку Редактировать, чтобы дать ему более содержательное описание.

Примечание:
Если установлен AiRBOOT, автоматическое обнаружение загрузочных томов может завершиться ошибкой, если новые тома были только что добавлены или удалены с помощью программы LVM, поскольку AiRBOOT обновляет свои таблицы томов только после перезагрузки.

Общие кнопки

Кнопка Применить сохраняет новый список в STYLER.INI.

Откатить восстанавливает список до состояния, в котором он был при открытии блокнота настроек.

Примечание:
В диалоговом окне расширенного выключения предустановлен параметр, и том для перезагрузки, который был выбран в последний раз, когда диалоговое окно было закрыто с помощью кнопки ОК.
