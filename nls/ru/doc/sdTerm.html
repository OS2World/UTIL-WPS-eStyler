При выключении системы некоторые программы работают некорректно и просто отказываются завершить работу, поэтому чтобы правильно закрыть систему необходимо использовать соответствующую утилиту, чтобы закрыть такие программы (в этом случае утилита должна быть внесена в список Программ выполняемых при выключении) или вернуться к функции Терминаторного списка, чтобы завершить их через API DosKillProcess().

Чекбокс Прервать работу следующих программ при выключении включает/отключает завершение работы перечисленных ниже программ.

По умолчанию, список программ, которые необходимо завершить содержит:

  • CHAT.EXE
  • CLKBASIC.EXE
  • CAD_POP.EXE
  • WPA_SUPPLICANT.EXE
Новые программы можно добавить, введя их имена в поле ввода под списком и нажав кнопку Добавить.

Убрать удаляет выбранную программу из списка.

При нажатии кнопки Редактировать имя выбранной программы вводится в поле ввода, чтобы пользователь мог его изменить. После изменения имени необходимо снова нажать кнопку Добавить, чтобы заменить имя выбранной программы новым содержимым поля ввода.

Найти открывает стандартный файловый диалог, позволяющий пользователю найти и выбрать имя исполняемого файла.

Применить сохраняет терминаторный список в файл STYLER.INI, а Откатить восстанавливает список до состояния, в котором он находился при открытии блокнота настроек.
