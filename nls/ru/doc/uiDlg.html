Обычно, когда программист явно не указал другой тип шрифта, во всех диалоговых окнах используется уродливый системный пропорциональный шрифт.

Стайлер позволяет переопределить это поведение системы, заставляя все диалоговые окна использовать шрифт предпочитаемый пользователем.
Когда установлен флажок чекбокса Установить шрифт по умолчанию для диалоговых окон все диалоговые окна будут использовать шрифт, выбранный с помощью кнопки выбора с многоточием.
Это будет работать во всех диалоговых окнах, в которых используется системный шрифт по умолчанию. Переопределить шрифты не по умолчанию заставляет все диалоговые окна использовать выбранный шрифт.

Примечание:
Элементы управления диалогового окна наследуют атрибут шрифта (параметр представления) из диалогового окна, если они не были явно настроены на использование другого шрифта. В последнем случае функция Стайлера по замене шрифтов работать не будет.
