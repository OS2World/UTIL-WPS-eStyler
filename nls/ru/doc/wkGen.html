Стайлер улучшает как внешний вид, так и поведение некоторых элементов управления OS/2 Presentation Manager и системы в целом.

Эти Клавиши Windows расширяют функциональные возможности клавиатуры, позволяя пользователю назначать некоторые полезные функции клавишам Windows на клавиатуре. Назначенные функции выполняются сразу после отпускания клавиши.

Не выполняйте никаких действий
Это просто отключает клавиши.
Скрыть/показать все окна
Это скрывает/отображает все окна, видимые на текущем рабочем столе (на случай, если Вы используете утилиту виртуального рабочего стола), что позволяет получить доступ к значкам рабочего стола.
Показать список окон
Эта функция отображает список окон в текущем положении курсора мыши.
Показать контекстное меню
Если в активном окне есть какое-либо контекстное меню, оно будет отображаться при отпускании выбранной клавиши.
Заблокировать десктоп
Блокирует рабочий стол, как соответствующий элемент всплывающего меню рабочего стола.
Открыть папку Накопители
Откроется папка "Накопители". Если папка уже открыта, она получает фокус и выводится наверх.
Открыть папку Настройка системы
Откроется папка "Настройка системы". Если папка уже открыта, она получает фокус и выводится наверх.
Открыть окно камандной строки OS/2
Откроется командное окно OS/2.
Показать меню десктоп WarpCenter
Откроется меню WarpCenter, в котором отображается текущее содержимое рабочего стола. Эта и следующие функции будут работать, только если WarpCenter в данный момент запущен.
Показать список окон WarpCenter
На экране отобразится список окон WarpCenter.
Показать список процессов WarpCenter
Если инструкция
SET KILLFEATUREENABLED=ON
присутствует в файле CONFIG.SYS, то этот список позволит Вам завершать процессы.
Показать список лотков WarpCenter
Эта функция отображает меню лотков WarpCenter.
Переход к следующему лотку WarpCenter
Следующий лоток WarpCenter перемещается наверх, как только будет отпущена выбранная клавиша.
Показать информационное меню WarpCenter
Эта функция показывает содержание меню Центра поддержки  с левой стороны от часов WarpCenter.
Показать меню XButton (сверху XCenter)
Меню кнопки X отображается, если XCenter установлен в верхней части экрана и содержит кнопку.
Показать меню XButton (снизу XCenter)
Меню кнопки X отображается, если XCenter установлен в нижней части экрана и содержит кнопку.
Показать список лотков XCenter
Отображает меню виджета на панели задач XCenter.
Переход к следующему лотку XCenter
Следующий лоток XCenter перемещается наверх, как только будет отпущена выбранная клавиша.
Показать XCenter лоток 1-10
Отобразится n-й лоток XCenter, предполагая, что лотки отсчитываются от верхней части меню виджетов лотка.
Переместить XCenter за/поверх других окон
Циклически перемещает XCenter назад или вперёд других окон
Кнопка Откатить восстанавливает текущие настройки до состояния, в котором они находились при открытии блокнота настроек.

Кнопки По умолчанию устанавливают для всех параметров значения по умолчанию.
