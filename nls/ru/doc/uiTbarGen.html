Общие настройки строки заголовка влияют на заголовок как в активном, так и в неактивном состоянии.

Разрешить улучшение строки заголовка подключает/отключает расширенные функции настройки заголовка.
Если этот параметр не отмечен, страницы настроек активные и неактивные окна будут недоступны.

Если чекбокс Переопределять параметры представления не установлен, заголовки, в которых не используются цвета по умолчанию для активного или неактивного состояния, будут игнорировать настройки Стайлера для фона заголовка.

Элемент управления Шрифт показывает название текущего шрифта заголовка. Кнопка выбора с многоточием отображает диалоговое окно шрифта по умолчанию, позволяющее выбрать другой шрифт.

С помощью раскрывающегося списка Выравнивание текста можно выровнять текст заголовка по центру или по левому краю.
